pagn is a Python module to solve the equations from Sirko and Goodman 2003 and Thompson et al. 2005 in a way that is simple, clear and highly customisable. Our aim is to provide a code that with the correct inputs provides a fully evolved AGN disk model through parametric 1D curves for key disk parameters such as temperature and density. We expect key applications of this code to be studies of migration torques in AGN disks, simulations of compact object formation inside gas disks, and comparisons with new, more complex models of AGN disks.
